The rel attribute

The rel attribute specifies the relationship between the current document and the linked document.

rel attribute values

  • noopener: 'Prevents the new page from being able to access the window.opener property, which can help prevent security vulnerabilities.'
  • noreferrer: 'Prevents the browser from sending the HTTP referer header to the new page, which can help protect user privacy.'

Images

The four main image formats are: JPEG, PNG, GIF, and SVG.

JPEG (Joint Photographic Experts Group) is a commonly used format for photographs and images with many colors.

PNG (Portable Network Graphics) is a format that supports lossless compression and transparency.

GIF (Graphics Interchange Format) is a format that supports animation and a limited color palette.

SVG (Scalable Vector Graphics) is a vector format that is resolution-independent and can be manipulated with CSS and JavaScript.
