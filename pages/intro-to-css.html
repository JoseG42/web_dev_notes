Cascading Style Sheets (CSS)

CSS is rules for styling web pages.

A selector is given a list of declarations. Declarations are separated by semicolons. Each declaration consists of a property and a value, separated by a colon.

Selectors

Selectors are used to target HTML elements for styling. They can be simple or complex, depending on the specificity needed.

  • Universal selector '*' This will select all elements on the page.

  • Type selector This will select all elements of a specific type, such as 'p' for paragraphs or 'h1' for headings.

  • Class selector This will select all elements with a specific class.

    Classes are defined in HTML with the 'class' attribute, and in CSS with a period (.) followed by the class name.

  • ID selector This will select a single element with a specific ID.

    IDs are defined in HTML with the 'id' attribute, and in CSS with a hash (#) followed by the ID name.

  • Grouping selector Group multiple selectors using a comma (,) to apply the same styles to different elements.

  • Chaining selectors Chain multiple selectors together by listing them with no spaces, to target specific elements.

  • Descendant combinator '.x .y' This will select all elements of type 'y' that are descendants of an element of type 'x'.

    represented by a space between the two selectors.

Properties

CSS properties are used to apply styles to elements. Each property has a name and a value, and they are written in a declaration block.

Some common CSS properties include:

  • Color Sets the color of text.

  • Background-color Sets the background color of an element.

  • Font-family Sets the font of text.

  • Font-size Sets the size of text.

  • Font-weight Sets the weight (boldness) of text.

  • Text-align Sets the alignment of text (left, right, center, justify).

  • Height Sets the height of an element.

  • Width Sets the width of an element.

  • Margin Sets the space outside an element.

  • Padding Sets the space inside an element.

Adding CSS to HTML

CSS can be added to HTML in three ways:

  • Inline CSS This involves adding CSS styles directly to an HTML element using the '< style >' attribute.

  • Internal CSS This involves adding a "< style >" element within the '< head >' section of the HTML document.

  • External CSS This involves linking to an external CSS file using a '< link >' element in the '< head >' section.
